Database information

dbType: Informe o tipo de base de dados a ser utilizado pelo modelo. É possível utilizar como base de dados um banco de dados Mysql ou Access. Também é possível criar um espaço celular virtual (grade de células). A base de dados pode ser construída utilizando um Sistema de Informações Geográficas (SIG), como, por exemplo, o TerraView.

Mysql

Database: Informe o nome do banco de dados utilizado no modelo.

Layer: Informe o layer do banco a ser utilizado.

Theme: Informe o tema do banco a ser utilizado.

Select: Informe as colunas da tabela de atributos a serem utilizadas pelo modelo.

Host: Informe o host do banco de dados a ser utilizado.

User: Informe o nome do usuário do banco de dados a ser utilizado.

Password: Informe a senha do usuário do banco de dados a ser utilizado.

Access

Access Database file: Selecione o arquivo do banco de dados a ser utilizado.

Layer: Informe o layer do banco a ser utilizado.

Theme: Informe o tema do banco a ser utilizado.

Select: Informe as colunas da tabela de atributos a serem utilizadas pelo modelo.

Virtual

xdim, ydim: Informe a dimensão do espaço celular a ser criado.
